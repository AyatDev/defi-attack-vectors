# DeFi Attack Vectors

Real-world DeFi attack analysis with exploit examples and mitigations.

## Attack Types

- **Flash Loan Attacks** — Price manipulation via uncollateralized loans
- **Reentrancy** — Recursive call exploits
- **Oracle Manipulation** — Price feed attacks
- **Front Running** — MEV and sandwich attacks
- **Rugpull** — Malicious liquidity removal
